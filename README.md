# LinkedIn Events Hub

> Elevating and expanding LinkedIn's event experience for seamless discovery and connection. Built for [Hackathon Name].

## The Vision
While LinkedIn already hosts events, discovering them currently relies heavily on manual searches. Our revamped **LinkedIn Gather** experience transforms this by introducing a prominent, single-click icon that takes you straight to a dedicated events page, supercharging it with new capabilities. Our solution turns event discovery into a proactive networking tool, helping professionals seamlessly track their schedules, discover local opportunities, and leverage mutual connections to make meaningful real-world interactions.

## Key Features

* **Dedicated 1-Click Access:** We introduced a prominent icon to the interface, giving users instant access to a dedicated events page and eliminating the friction of manual event searches.
* **Intelligent Attendee Insights:** RSVP to events and instantly view the guest list with a priority focus on your network. The UI explicitly highlights mutual connections (e.g., *"80 people going, 25 connections"*), allowing for targeted networking.
* **Integrated Calendar & Conflict Prevention:** All RSVPs automatically sync to a built-in event calendar. The system tracks your itinerary and actively prevents scheduling overlaps.
* **Rich Event Details:** Clicking any event on your calendar opens a comprehensive modal featuring the event banner, full details, and the curated attendee list.
* **Smart Networking Suggestions:** The platform analyzes overlapping event attendance and proactively suggests new connections based on how many events you have attended with the same person.
* **AI-Powered Follow-Ups:** Post-event nudges remind you to reach out and solidify connections with people you recently attended an event with, complete with AI-suggested messages to help initiate the conversation naturally.
* **Personalized Event Discovery:** An AI-driven recommendation engine suggests upcoming events tailored to your professional profile—factoring in your courses, skills, and industry, while primarily highlighting events in your current location.
